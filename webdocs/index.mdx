## ğŸ¯ What is AIKit?

AIKit is a **universal package manager** for AI agent extensions that enables developers to:

- **Create reusable AI commands** and templates across different AI assistants
- **Share AI tools** through a decentralized package ecosystem
- **Integrate seamlessly** with 17+ AI coding assistants (Claude, Cursor, Copilot, Gemini, and more)
- **Streamline workflows** with standardized project initialization and package management
- **Foster community collaboration** through open-source AI tool sharing

### Quick Example

```bash
# Install AIKit
cargo install aikit

# Create your first AI-powered project
aikit init my-ai-project --ai claude

# Install community packages
# Browse packages at https://github.com/search?q=aikit&type=repositories
aikit install owner/code-review-tools

# Start developing with AI assistance
cd my-ai-project
# Your AI agent now has access to code review tools!
```
