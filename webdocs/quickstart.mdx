# AIKit Quickstart

Get up and running with AIKit in three simple steps. This guide will have you managing AI agent packages and creating AI-powered projects in under 10 minutes.

## Step 1: Install AIKit

<AccordionGroup>
  <Accordion icon="download" title="Download from GitHub Releases">
    Visit the [AIKit releases page](https://github.com/goaikit/aikit/releases) and download the latest version for your platform:

    - **Linux**: Download `aikit-x86_64-unknown-linux-gnu.tar.gz`
    - **macOS**: Download `aikit-x86_64-apple-darwin.tar.gz`
    - **Windows**: Download `aikit-x86_64-pc-windows-msvc.zip`

    Extract the archive and place the `aikit` binary in your PATH.
  </Accordion>

  <Accordion icon="terminal" title="Verify Installation">
    Open a terminal and run:

    ```bash
    aikit --version
    ```

    You should see output similar to:
    ```
    AIKit 0.1.19
    ```
  </Accordion>
</AccordionGroup>

## Step 2: Configure AI Agents

<AccordionGroup>
  <Accordion icon="cog" title="Set up your first AI agent">
    Initialize AIKit in your project:

    ```bash
    # Create a new project
    aikit init my-ai-project --ai claude

    # Or set up in existing directory
    cd existing-project
    aikit init --here --ai cursor
    ```

    This creates the necessary configuration files and agent-specific directories.
  </Accordion>

  <Accordion icon="key" title="Configure GitHub Token (Recommended)">
    For full functionality, set up a GitHub personal access token:

    ```bash
    # Create token at: https://github.com/settings/tokens
    # Required scopes: repo, read:packages
    export GITHUB_TOKEN="your_token_here"
    ```
  </Accordion>
</AccordionGroup>

## Step 3: Start Using AIKit

<AccordionGroup>
  <Accordion icon="play" title="Test your setup">
    Verify everything is working:

    ```bash
    # Check available tools
    aikit check

    # Browse community packages
    # Visit https://github.com/search?q=aikit&type=repositories

    # Install a useful package
    aikit install owner/code-review-tools

    # List your installed packages
    aikit list
    ```
  </Accordion>

  <Accordion icon="package" title="Create your first package">
    Build and share your own AI tools:

    ```bash
    # Create a new package
    aikit package init my-tools --description "My custom AI tools"

    # Add your commands (edit the generated files)

    # Build the package
    aikit package build

    # Publish to GitHub
    aikit package publish yourusername/my-tools
    ```
  </Accordion>
</AccordionGroup>
